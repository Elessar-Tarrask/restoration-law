# Услуга 1: Восстановление платежеспособности

## Описание услуги

Восстановление платежеспособности — это процедура, применяемая в судебном порядке, направленная на восстановление финансового состояния должника без присвоения статуса банкрота. Процедура позволяет гражданам со стабильным доходом получить рассрочку на оплату долгов до 5 лет.

## Статус
✅ **Первая услуга** - Документация уже подготовлена

## Основные этапы

1. Первичная консультация (1-2 дня)
2. Сбор документов (5-10 дней)
3. Подготовка заявления (2-3 дня)
4. Подача в суд (1 день)
5. Рассмотрение судом (10 рабочих дней)
6. Назначение финансового управляющего (3-5 дней)
7. Разработка плана восстановления (15-30 дней)
8. Утверждение плана судом (10-15 дней)
9. Исполнение плана (до 5 лет)
10. Завершение процедуры (10-15 дней)
